# CobaCS3 - Ekstensi Pribadi CloudStream

Kumpulan ekstensi *private* untuk CloudStream (Sflix, Dubbindo, Oppadrama, dan Oploverz).

## Cara Install (Pilih Salah Satu)

### 1. Install via Repo (Semua Ekstensi Sekaligus)
Salin link berikut dan tambahkan di CloudStream → Settings → Extensions → Add repository:

```
https://raw.githubusercontent.com/JaTri19/cobacs3/builds/repo.json
```

---

### 2. Install Manual per Ekstensi
Jika repo di atas tidak bisa diakses, unduh file ekstensinya satu-per-satu lewat tautan di bawah (pastikan sudah login akun GitHub di browser HP):

* 📥 [Install Sflix](https://raw.githubusercontent.com/JaTri19/cobacs3/builds/Sflix.cs3)
* 📥 [Install Dubbindo](https://raw.githubusercontent.com/JaTri19/cobacs3/builds/Dubbindo.cs3)
* 📥 [Install Oppadrama](https://raw.githubusercontent.com/JaTri19/cobacs3/builds/Oppadrama.cs3)
* 📥 [Install Oploverz](https://raw.githubusercontent.com/JaTri19/cobacs3/builds/Oploverz.cs3)

*(Setelah file ter-download, cukup ketuk filenya di menu Notifikasi / File Manager Anda untuk memasukkannya secara otomatis ke CloudStream).*
