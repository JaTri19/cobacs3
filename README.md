# CobaCS3 - Ekstensi Pribadi CloudStream

Kumpulan ekstensi *private* untuk CloudStream (Sflix, Dubbindo, Oppadrama, dan Oploverz).

## Cara Install (Pilih Salah Satu)

### 1. Install via Repo (Semua Ekstensi Sekaligus)
Salin link berikut dan tambahkan di CloudStream → Settings → Extensions → Add repository:

```
https://raw.githubusercontent.com/JaTri19/cobacs3/builds/repo.json
``
